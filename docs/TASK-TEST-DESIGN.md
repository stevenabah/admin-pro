基于 PRD 文档《企业级任务管理系统》编写，聚焦任务创建、分配、跟踪、统计四大核心功能。

## 1. 任务创建功能测试用例

### TC-TASK-C001: 正常创建任务

```
用例编号: TC-TASK-C001
标题: 使用完整信息创建任务成功
模块: 任务创建
优先级: P0
前置条件: 已登录，用户有任务创建权限
测试步骤:
  1. 点击"创建任务"按钮
  2. 输入标题: "完成用户模块开发"
  3. 输入描述: "开发用户管理相关功能"
  4. 选择优先级: 高 (P0)
  5. 设置截止日期: 2026-04-01
  6. 上传附件: test.pdf (5MB)
  7. 点击"创建"按钮
预期结果:
  - 任务创建成功，提示"任务创建成功"
  - 跳转至任务详情页
  - 任务状态默认为"待办"
  - 创建人为当前用户
  - 任务列表中显示新任务
```
